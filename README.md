# 🌌 Starforge Portfolio Site

This repository contains the development of my portfolio website. It currently includes the completed website style guide and the early structural outline of the portfolio itself. The project is in active development and will continue to be expanded and refined over the next few months. 

## ⚡ Current Status

This project is a work in progress.

### ✔ Completed so far:
- Website style guide
- Color palette 
- Button styling and interaction states
- Initial portfolio structure

### ⏲ In progress:
- Homepage layout
- Featured project sections
- Dedicated project pages

### 🧠 Planned:
- Vega project page
- Sister Aurora project page
- DevHandbook project page
- Final portfolio polish and responsive improvements

## 🎯 Purpose

The goal of this project is to build a clean, structured portfolio site, while practicing HTML and CSS fundamentals such as: layout systems, reusable design patterns, and iterative front-end development. In addition, it will also include more interactive elements developed through JavaScript as well. As of now, everything in the repository is built with CSS and HTML.

## ⌛ Development Timeline

Currently, this project is being developed incrementally, and is expected to be completed by the end of August 2026!

## 📝 Notes

This repository reflects my active learning and building process. The completed style guide serves as the design foundation for the rest of the site, as the project continues to grow; however, nothing is set in stone, and some elements may change and evolve as I learn and improve over these next few months!
